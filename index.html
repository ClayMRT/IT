Жергілікті желі және IP-адрестеу

LAN IP Wireshark Қорытынды Қолдау

Қарапайым жергілікті желіні орнату және талдау

Бұл бет жергілікті желіні орнату, IP-адрестеу және Wireshark көмегімен талдау тәсілдерін көрсетеді.

1. LAN орнату

LAN (Local Area Network) — бірнеше компьютерді біріктіріп, ортақ деректермен және интернетпен бөлісуге арналған желі.

  • Switch немесе Router арқылы байланыс орнату
  • Ethernet немесе Wi-Fi пайдалану
  • Бір IP диапазонында болу
Топология:
HostA (192.168.1.10) --- Switch --- HostB (192.168.1.11)
                         |
                      Router (192.168.1.1)

2. IP-адрестеу

Әр құрылғыға IP беру — олардың бір-бірін табуы үшін қажет.

  • Network: 192.168.1.0/24
  • Router: 192.168.1.1
  • Hosts: 192.168.1.10, 192.168.1.11
Windows:
netsh interface ipv4 set address name="Ethernet" static 192.168.1.10 255.255.255.0 192.168.1.1

Linux:
sudo ip addr add 192.168.1.10/24 dev eth0
sudo ip route add default via 192.168.1.1

3. Wireshark және трафик талдау

Wireshark — желідегі пакеттерді нақты уақытта көріп, талдауға мүмкіндік беретін құрал.

  • ARP: arp
  • ICMP (ping): icmp
  • HTTP сұраныстар: http.request.method == "GET"
  • DNS: dns
sudo tcpdump -i eth0 host 192.168.1.10 -w capture.pcap

4. Қорытынды

Жергілікті желі — желілік технологияларды түсінудің негізі. IP-адрестеу және Wireshark көмегімен желіні тиімді талдауға болады.

5. Қолдау

Егер сізде сұрақтар немесе ұсыныстар болса — төмендегі форманы толтырыңыз.
